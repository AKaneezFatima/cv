Horizontal Rule(Self-closing tag )

Attributes of hr tag

1. align - Sets the color of the rule through color name or hexadecimal value.

2. noshade - Sets the rule to have no shading.

3. size - Sets the height, in pixels, of the rule.

4. width - Sets the length of the rule on the page through a pixel or percentage value.
